# Hi, I'm Jacob Surber

**Floor Planning Analyst** at P.L. Marketing — building browser-based tooling and automation that support Kroger retail merchandising and space-planning decisions.

BS in Marketing and Management from Mount St. Joseph University (3.9 GPA, Academic All-HCAC Award).

---

## What I Do

**Floor Planning & Space Optimization** — Lead store layout planning and remodel strategy for a high-volume grocery retail program: block layouts, test-fit plans, and planogram assignment validation against corporate merchandising standards. Drove a 12% sales increase in a high-priority store through strategic space optimization.

**Internal Tooling** — Design and build browser-based decision-support tools that replace manual floorplan and planogram review — validating commodity footage and initiative applicability, checking bakery floorplans against fixture/POG standards, and diffing old vs. new store mappings (footage, aisle, orientation, version) through Excel/SheetJS workflows.

**Automation & Validation** — Convert merchandising and floorplan compliance rules into machine-readable validation logic, automating space-planning checks across PDFs, scanned images, CSVs, and Excel exports to catch version mismatches and fixture gaps that used to require manual review.

**AI-Assisted Development** — Use Claude Code and generative AI to design, build, and ship internal tooling faster — directing and reviewing AI-generated code rather than hand-rolling every line, while staying responsible for correctness, edge cases, and how the tools handle data.

---

## Featured Project

**Retail Merchandising Tools** — A suite of five browser-based tools automating retail floorplan and merchandising compliance checks: POG version validation against Category Index scores, floorplan/fixture compliance checking with an OCR fallback for scanned plans, store-layout diffing (footage/aisle/orientation), and fuzzy natural-language spreadsheet search. Fully client-side — no backend, nothing leaves the browser. *(Private repo — happy to walk through the code directly.)*

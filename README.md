## What I Do

- 🏬 Lead floor planning initiatives — store layout design, block layouts, test-fit plans, and remodel strategy, aligned to corporate standards
- 🛠️ Design and build browser-based decision-support tools for validating planogram assignments, commodity footage, and store-specific initiative applicability
- 📋 Built a browser-based Bakery Standup Checker to validate bakery floorplans against fixture and POG standards
- 📊 Automate retail space-planning checks across PDFs, images, CSVs, and Excel files, converting manual merchandising/compliance review into machine-readable validation logic

- 🤖 Use Claude Code and generative AI to prototype and build internal tooling faster
